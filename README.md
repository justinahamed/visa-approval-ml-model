# Visa Approval Prediction Model

## Overview
Built machine learning models to predict visa approval outcomes and identify key influencing factors.

## Problem
Visa approval decisions depend on multiple applicant features. The goal was to improve prediction accuracy and understand key drivers.

## Approach
- Data preprocessing and feature engineering
- Trained multiple models: Random Forest, Gradient Boosting, XGBoost, Stacking
- Hyperparameter tuning using GridSearchCV
- Model evaluation using classification metrics

## Results
- Improved predictive performance using ensemble methods
- Identified important features influencing approval decisions

## Tech Stack
Python, pandas, scikit-learn, XGBoost
